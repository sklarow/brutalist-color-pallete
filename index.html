Brutalist Color Palette Maker

Brutalist web design embraces raw, unpolished aesthetics with bold typography, stark contrasts, and functional layouts that prioritize content over decoration. This color palette maker generates harmonious color schemes using the golden ratio (φ ≈ 1.618), creating mathematically balanced palettes perfect for brutalist interfaces. Each palette includes base colors, golden ratio-based hue rotations, and carefully desaturated variants—all designed to work together with maximum visual impact and minimal design complexity.

Palette Rules (Golden Ratio Based)

  • Choose your base color
  • Hue rotations: 137.5° (360°/φ²), 222.5° (360°/φ), 275° (137.5°×2)
  • Saturation: 60% desat, 38.2% desat (golden ratio), 23.6% desat
  • Lightness: ±38.2% adjustments using golden ratio
